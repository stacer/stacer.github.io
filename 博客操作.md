#	博客恢复操作:

##	一、git操作

*	1.`git clone hexo` 复制分支到本地blog目录

*	2.配置全局git个人信息
`git config --global user.name "stacer"`
`git config --global user.email "stacer@126.com"`

*	3.生成本地密钥并配置到github
`ssh-keygen -t ras -C "stacer@126.com"`

*	4.测试本地与github的通信
`ssh -T git@github.com`


##	二、blog的配置
*	1.安装nodejs
*	2.配置淘宝npm
*	3.安装hexo
`cnpm install -g hexo-cli`

*	4.安装依赖库
`cnpm install`

*	5.安装部署插件
`cnpm install hexo-deployer-git`

*	6.安装搜索插件
`cnpm install --save hexo-generator-search`

##	三、发布博客的操作
*	1. `hexo clean #清除发布目录`
*	2. `hexo new #新建博客`
*	3. `hexo g #生成静态文件`
*	4. `hexo server #本地发布`
*	5. `hexo d #同步github发布(前提需要配置好_config.yml中的git信息)`

##	四、同步整个博客到github
*	1.`git checkout -b hexo #切换到hexo分支`
*	2.`git add .	#添加所有文件到git仓库`
*	3.`git commit -m ""	#提交文件到本地git仓库`
*	4.`git push origin hexo  #推送文件到github上`
